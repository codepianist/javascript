## SVG Objects
- XML based vector image
- Came with HTML5 to draw shapes. Can use to render an image
- Raster vs. Vector

| Raster | Vector |
|------------------------|--------------------------------------------------|
| .jpeg,.gif,.png | .svg |
| Pre-defined resolution | Can scale |
| Don't scales, distorts | It scales, renders accordingly to the resolution |
